A Game Engine is is a software framework which is designed to help developer to create video game. Some core functionalities provided by a game engine are typically: - rendering engine (load/display/animate 2d image, 3d models) - a physic engine (simulate gravity, wind, mouvement of a bounced ball by example) - collision detection (when you walk into a wall, you should be stop and not go throught it) - artificial intelligence (to simulate ennemies' behavior) - networking (provide easy way to develop multiplayer game) - sound, music - an avantage of using game engine is this reusability, and possibility to port game to another platform ---------------------------------------------------------------------------------------------------------- Here is a table which compare some major html5 game engine available on the market As you can see,
